# Xot Module Documentation - Consolidated

This directory contains consolidated and organized documentation for the Xot module, which serves as the core foundation of the Laravel modular monolith application.

## Directory Structure

- **/architecture** - System architecture and development rules
- **/best_practices** - Coding standards and quality guidelines
- **/module_development** - Module creation and structure guides
- **/phpstan** - PHPStan static analysis configuration and rules
- **/testing** - Testing strategies and best practices
- **/actions** - Action patterns and usage guidelines
- **/filament** - Filament PHP admin panel documentation
- **/standards** - Coding standards and conventions
- **/archive** - Historical and archived documentation
- **/guides** - General guides and README files

## Key Documentation Files

### Core Standards
- `CODE_QUALITY.md` - Code quality standards
- `XOTBASE_EXTENSION_RULES.md` - Xot base class extension rules
- `TRAIT_METHOD_SIGNATURE_RULES.md` - Trait method signature conventions
- `SVG_ICON_STANDARDS.md` - SVG icon standards

### Module Development
- `MODULE_CONFIGURATION_BEST_PRACTICES.md` - Module configuration guidelines
- `PANEL_PROVIDER.md` - Panel provider implementation

## Usage

This consolidated structure organizes the previously scattered 1000+ documentation files into logical categories for easier navigation and maintenance.

## Original Structure

The original documentation was located in `Modules/Xot/project_docs/` with numerous subdirectories and archived content. This consolidation preserves all content while improving organization.
