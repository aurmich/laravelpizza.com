Comandi MCP

mcp tailwind analyze ./resources/css/app.css

mcp vite build --watch

mcp component create pizza-card

Crea un nuovo componente pizza-card

Configurazione MCP

MCP_SERVER_FILESYSTEM_ENABLED=true
MCP_SERVER_TAILWIND_ENABLED=true
MCP_SERVER_VITE_ENABLED=true
MCP_TAILWIND_CONFIG_PATH=./tailwind.config.js
MCP_VITE_CONFIG_PATH=./vite.config.js
MCP_COMPONENT_SCAN_PATH=./resources/views/components
MCP_CSS_ANALYSIS_ENABLED=true
MCP_ASSET_OPTIMIZATION=true

Esempi di Integrazione MCP
